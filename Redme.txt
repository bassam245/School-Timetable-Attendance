# مشروع إدارة الجدول الدراسي والحضور - C#

ده مشروع Console Application معمول بلغة #C لإدارة الحصص المدرسية وتسجيل الحضور.

## مميزات المشروع:

- إضافة حصص وعرض الجدول الدراسي
- ترتيب الحصص حسب وقت البداية (Selection Sort)
- البحث عن مادة باستخدام Linear Search
- تسجيل حضور الطلاب باستخدام Queue يدوي
- تطبيق مبادئ OOP: الوراثة، الكلاسات المجردة، الإنترفيس، الـ enums

## الملفات الرئيسية:

- `Person.cs` → الكلاس الأساسي لكل الأشخاص
- `Student.cs` و `Teacher.cs` → الوراثة من Person
- `TimetableEntry.cs` → كلاس الحصة الدراسية
- `Timetablemange.cs` → مسؤول عن إضافة وفرز وبحث الحصص
- `AttendanceQueueManager.cs` → إدارة حضور الطلاب
- `Program.cs` → البرنامج الرئيسي
